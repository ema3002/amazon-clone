# 🛒 Amazon Clone

A static front-end clone of the Amazon homepage built with pure **HTML** and **CSS**.

## 📸 Preview

> Open `index.html` in your browser to see the live preview.

## 🗂️ Project Structure

```
AMAZON_CLONE/
│
├── image/               # All images folder
│   ├── image_logo.png
│   ├── image_hero.jpg
│   ├── image_box1.jpg
│   └── ...
│
├── index.html           # Main HTML file
└── style.css            # All styles
```
## ✨ Features

- ✅ Responsive Navbar with logo, search bar, sign-in & cart
- ✅ Delivery location indicator
- ✅ Secondary navigation panel (deals, customer service, etc.)
- ✅ Hero banner section
- ✅ Product category cards (Kitchen, Home, Gaming, Fashion)
- ✅ Footer with 4 panels — links, logo, copyright

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & layout |
| CSS3 | Styling, Flexbox, Grid |
| Font Awesome 7 | Icons (cart, location, search) |

## 🚀 How to Run Locally

1. Clone the repository:
```bash
   git clone https://github.com/YOUR_USERNAME/amazon-clone.git
```
2. Open the folder and double-click `index.html`  
   **OR** use VS Code Live Server extension for hot reload.

## 📌 Notes

- This is a **UI clone only** — no backend, no real purchasing functionality.
- Built for learning and portfolio purposes.
- All images should be placed in the root folder alongside `index.html`.
